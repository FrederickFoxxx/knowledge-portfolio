# flat

```
const arr = [1, 2, [3, 5]]

const flatArr = arr.flat();

console.log(flatArr)
// [ 1, 2, 3, 5 ]
```

# flatMap 

```
const arr = [1, 2, 3, 4]

arr.flatMap(x => [x, x * 2]);
```
